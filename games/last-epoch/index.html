📖 About Seasons
Last Epoch is an action RPG developed by Eleventh Hour Games, combining time travel, exciting dungeon crawling, and deep character customization. The game features an accessible crafting system, a deep skill tree, and a robust endgame. Seasons (known as Cycles) run for approximately 3 to 4 months, offering fresh content, competitive ladders, and item updates.

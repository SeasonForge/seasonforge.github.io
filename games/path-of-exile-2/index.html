📖 About Seasons
Path of Exile 2 is a next-generation action RPG developed by Grinding Gear Games. Set years after the original game, it introduces a brand new campaign, reworked class mechanics, a new skill gem system, and improved graphical fidelity. It shares a co-existing endgame and microtransactions with the original Path of Exile, with seasons running concurrently.

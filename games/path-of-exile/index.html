📖 About Seasons
Path of Exile is a free-to-play action RPG developed and published by Grinding Gear Games. The game features a dark fantasy world, deep character customization, and a complex passive skill tree. Seasons (known as Leagues) run for approximately 3 to 4 months, introducing fresh league mechanics, balance updates, and economy resets.
